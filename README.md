# משימה דחופה 2050 - הקוד להצלת חוף הכרמל

חדר בריחה דיגיטלי לשבוע הסביבה תשפ"ו - תיכון כפר גלים, כיתה י'.

## על הפרויקט

לומדה דיגיטלית אינטראקטיבית באורך של כ-40 דקות, המשלבת בין תכנים פדגוגיים בנושא משבר האקלים לבין מנגנון של חדר בריחה.

**השלבים בלומדה:**

1. **תדרוך פתיחה** - מצגת על תזונה בת קיימא ומשבר האקלים
2. **הודעה ממאדים** - סרטון של ד"ר גלי כפר, בוגרת בית הספר שכעת חיה במאדים בשנת 2050
3. **תדרוך משימה** - הצגת ארבעת חדרי הבריחה
4. **חדר 1 - טביעת הרגל הסודית** - תעשיית האופנה המהירה
5. **חדר 2 - תחנת ניטור הכרמל** - שרשרת הגורמים לעליית מפלס הים והתערבויות אקלים
6. **חדר 3 - תפריט 2050 לכפר גלים** - תזונה בת קיימא בפנימייה
7. **חדר 4 - מטה החקלאות החכמה** - חקלאות מדייקת באמצעות רחפנים
8. **חדר 5 - מרכז שליטה לשיקום** - הפעלת ארבעת מודולי השיקום
9. **תעודת ההצלחה** - תעודה אישית להורדה

## טכנולוגיות

- HTML, CSS, JavaScript (קובץ יחיד עצמאי)
- Tailwind CSS דרך CDN
- html2canvas להורדת התעודה
- Google Fonts (Heebo, Space Grotesk)
- Material Symbols Outlined

## הפעלה מקומית

פותחים את `index.html` בכל דפדפן מודרני. אין צורך בהתקנה או build.
